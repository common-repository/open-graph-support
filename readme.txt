== Description ==

Makes your content appealing in Facebook and Google+ by enabling Open Graph meta data.

All you have to do is install and activate the plugin. Open Graph data such as title, description, and image is added automatically to your pages.

Possibility to use Facebook App ID to get insights of your site's social media presence.

This tool is part of the Poor Man's [WordPress SEO](http://mokimoki.net/poor-mans-wordpress-seo/) plugin that bundles a wide variety of important SEO features.
